# Java İlk Program

Java ile ekrana merhaba yazan program:

```
package com.levham.deneme;// <- bulunduğu klasör
public class Deneme { // <- dosyanın bulunduğu sınıf

    public static void main(String[] args) { //<-- ana fonksiyonumuz
        System.out.println("Merhaba"); // <- ekrana yazı yazdıran kod
    }
}
```
