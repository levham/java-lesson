Java Anahtar Kelimeler Listesi 
1. Temel Yapılar
class → Sınıf tanımlamak için
interface → Arayüz tanımlamak için
enum → Sabit değerler kümesi tanımlamak için
package → Paket tanımlamak için
import → Paket veya sınıf içeriğini kullanmak için

2. Değişkenler ve Veri Tipleri
int, double, float, char, boolean, byte, short, long → Temel veri tipleri
String (anahtar kelime değil ama özel sınıf) → Metin tutmak için
final → Sabit değer veya değiştirilemez sınıf/metot/değişken
static → Sınıfa ait üyeler tanımlamak için
volatile → Çoklu thread erişiminde değişkenin güncel değerini korumak için

3. Kontrol Yapıları
if, else → Koşul ifadeleri
switch, case, default → Çoklu seçim yapısı
for, while, do → Döngüler
break → Döngüyü veya switch’i sonlandırır
continue → Döngünün bir sonraki adımına geçer
return → Metottan değer döndürür

4. Hata Yönetimi
try, catch, finally → Hata yakalama ve temizleme
throw → Hata fırlatmak için
throws → Metot imzasında hata türlerini belirtmek için

5. Nesne Yönelimli Programlama
new → Nesne oluşturmak için
this → Mevcut nesneyi işaret eder
super → Üst sınıfı işaret eder
extends → Kalıtım için
implements → Arayüz uygulamak için
abstract → Soyut sınıf/metot tanımlamak için
synchronized → Thread güvenliği sağlamak için

6. Erişim Belirleyiciler
public → Her yerden erişilebilir
protected → Aynı paket + alt sınıflardan erişilebilir
private → Sadece bulunduğu sınıfta erişilebilir
default (yazılmaz, paket içi erişim)
